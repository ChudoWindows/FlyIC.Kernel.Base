# FlyIC.Kernel.Base - базовая библиотека ядра

## Описание
FlyIC.Kernel.Base - библиотека ядра платформы, предоставляет все базовый типы (INT*, UINT*, указатели, т.п.), также предоставляет инструменты для конвертации символов из одной кодировки Unicode в другую

## Состав
### Заголовки
1. Types.hpp - базовые типы
2. ExChar.hpp - расширенный символ в формате UTF-32
### Классы
1. ExChar - расширенный символ в формате UTF-32
### Типы
1. INT8 - Знаковый целочисленный тип чисел, 8 битный
2. INT16 - Знаковый целочисленный тип чисел, 16 битный
3. INT32 - Знаковый целочисленный тип чисел, 32 битный
4. INT64 - Знаковый целоцисленный тип чисел, 64 битный
5. UINT8 - Беззнаковый целочисленный тип чисел, 8 битный
6. UINT16 - Беззнаковый целочисленный тип чисел, 16 битный
7. UINT32 - Беззнаковый целочисленный тип чисел, 32 битный
8. UINT64 - Беззнаковый целоцисленный тип чисел, 64 битный
9. Указатели на целочисленный типы: LPINT8, LPUINT64. Префикс "LP" к оригинальному названию типа
10. Константные указатели на целочисленные типы: LPCINT32, LPCUINT8. Префикс "LPC" к оригинальному названию типа
11. HANDLE - хэндлы
12. LPVOID, LPCVOID - неопределенный указатель
